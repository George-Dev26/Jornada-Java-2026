📚 Jornada Java 2026: Do Caderno ao Código
🎯 Sobre o Projeto
Este repositório é o registro da minha transição de carreira para o desenvolvimento Java.
O diferencial deste projeto é que ele é o resultado de um método de aprendizado ativo.

Cada algoritmo presente aqui foi primeiro estudado teoricamente e registrado manualmente em cadernos,
garantindo a absorção dos conceitos fundamentais antes da implementação prática no IntelliJ IDEA.

🛡️ Pilares de Estudo
O projeto está organizado para refletir minha evolução nos seguintes temas:

[x] Lógica de Programação: Variáveis, operadores e estruturas condicionais.

[x] Estruturas de Repetição: Otimização de processos com loops.

[x] Orientação a Objetos (POO): Classes, atributos, métodos e construtores.

[ ] Poder da POO: Herança, Polimorfismo, Classes Abstratas e Interfaces (Em transição).

[ ] Especialização: Collections (List/Map), Exceptions e Enums.

🛠️ Tecnologias e Ferramentas
Java 21 (LTS)

IntelliJ IDEA Community Edition

Git & GitHub
